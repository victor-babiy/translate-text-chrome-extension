## Старт разработки
- npm i
- npm run dev

## Сборка проекта
- npm run build

### Общая информация
Для паблиша екстеншина в Chrome собираем проект командой `npm run build`, после папку `build` заливаем в chrome://extensions или добавляем в Chrome Store.
